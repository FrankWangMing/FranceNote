# Agent 说明

本仓库前端由 `client/` 下的 React + TypeScript 应用构成。

## 开发约定

- 开发时遵循 **`.cursor/rules/`** 中的前端规则：`frontend-ts-react.mdc`、`frontend-ui-tailwind.mdc`、`frontend-a11y.mdc`。
- 新建组件、新建页面、做可访问性检查时，优先按 **`.cursor/skills/frontend-workflows`** 中的 Task plans 执行。

## 技术栈

React 19、TypeScript、Vite、Tailwind CSS v4、Radix UI（components/ui）、wouter、framer-motion、react-hook-form + zod。
